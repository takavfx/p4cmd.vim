# p4cmd.vim

Perforce commands support for Vim.

## Installation

dein.vim

```toml
[[plugins]]
repo = 'takavfx/dailynote.vim'
depends = 'denops.vim'
```

lazyvim

```lua
return {
  {'vim-denops/denops.vim'},
  {
    'takavfx/p4cmd.vim',
    depends = 'denops.vim',
  },
}
```
